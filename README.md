# Eco Parv - Save Nature 🌱

Welcome to **Eco Parv**, a comprehensive ecosystem designed to bridge the gap between unused resources and those in need while promoting a sustainable, eco-friendly lifestyle.

---

## 1. Chosen Vertical
**Sustainability & Community Welfare**  
Eco Parv addresses environmental waste and social inequality by connecting donors with NGOs, promoting recycling through startups, encouraging eco-friendly product adoption, and mobilizing volunteer teams for community action.

---

## 2. Approach and Logic
Our approach is centered on **Practical Action and Community Connectivity**.  
Key integrations include:
- **Donation Portal** with geolocation-based pickup.  
- **Volunteer & NGO Connect** using local storage for instant search.  
- **Renew & Reuse Routing** to real recycling startups.  
- **Smart Assistant** for navigation and FAQs.  
- **Eco-Friendly Products** via Google Form → Apps Script → frontend rendering.  
- **Authentication & Ownership** using Firebase Authentication.  
- **Dashboards** for donors and NGOs with category-wise actions.  
- **Real-Time Updates** powered by Firestore.  
- **Impact Dashboard** to track carbon saved and community help metrics.

---

## 3. How the Solution Works
A lightweight, frontend-driven web application built with HTML, CSS, and JavaScript, enhanced with Firebase services:
- **Authentication** for secure sign-in/sign-up.  
- **Dynamic Dashboards** for donors and NGOs with live Firestore sync.  
- **Impact Dashboard** visualizes eco-impact (carbon saved, people helped).  
- **Responsive UI** across devices.  
- **Smart Eco Assistant** for instant guidance.  
- **Eco Products Integration** with Meesho redirect.

---

## 4. Assumptions Made
- **Client-Side Storage** sufficient for prototype volunteer/donation listings.  
- **Email Client Availability** for `mailto:` forms.  
- **Geolocation Permissions** granted for address resolution.  
- **Static Assets** used for demo purposes.
